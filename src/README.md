# Project Structure

```
src/
├── environment.rs // Handles Tron environment setup and management
├── expr.rs // Handles expressions in Tron
├── interpreter.rs // Handles Tron interpreter
├── main.rs // Entry point of the Tron program
├── natives.rs // Handles Tron native functions
├── parser.rs // Handles parsing of Tron code
├── resolver.rs // Handles resolution of Tron code
├── scanner.rs // Handles scanning of Tron code
├── stmt.rs // Handles statements in Tron
└── tstd.rs // Contains Tron natives
```
